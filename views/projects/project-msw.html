Search

New Search Interface and Features

Redesigned enterprise search with the same design elements as MSW. Added new faceted layout and federations along with new media and content types that we now supported. Microformats and schema where added to items in order to increase searchability and portability of content objects alongside the metadata driven and FAST search implementations. As MySite came online, the proliferation of hashtags made its way to our designs so that we supported them in the same manner as enterprise keywords, subject to company specific ontologies.

Search Result Threading

Conversational results get threading for context

Pinklines for new community search scope displaying the threading feature provided for contextual relevance. This was an incredibly critical feature as we decided to crawl and display all types of conversational results like distribution lists, forum threads and our company wide activity stream. A good deal of mapping across all the meta data ensured that we ended up with a solid and intuitive solution.
